# Homework 04 – Discriminative vs. Generative Models

## Course

Artificial Intelligence (Spring 2025)


---

## Overview

This assignment introduces students to probabilistic and discriminative machine learning models through mathematical analysis and complete algorithm implementation from scratch.

Students compare Gaussian classifiers, Gaussian Naive Bayes, Logistic Regression, LDA, and QDA on a real-world cardiovascular dataset.

---

## Learning Objectives

- Logistic Regression
- Newton Optimization
- Maximum Likelihood Estimation
- Gaussian Generative Models
- Gaussian Naive Bayes
- Linear Discriminant Analysis (LDA)
- Quadratic Discriminant Analysis (QDA)
- Decision Boundary Analysis
- Model Robustness
- Clinical Risk Prediction
