# Homework 02 – Informed Search, Constraint Satisfaction Problems, and Adversarial Search

## Overview

This assignment was designed as part of the undergraduate Artificial Intelligence course.

Unlike traditional programming assignments, this project integrates multiple core areas of Artificial Intelligence into two large-scale scenarios requiring algorithm design, modeling, implementation, and experimental evaluation.

Students are expected to implement all algorithms from scratch without using external AI libraries.
