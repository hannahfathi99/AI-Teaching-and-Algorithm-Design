# Homework 03 – Linear and Nonlinear Regression

## Course

Artificial Intelligence (Spring 2025)

---

## Overview

This assignment was designed to strengthen students' understanding of regression models through both theoretical analysis and practical implementation.

Students are required to derive mathematical formulations, implement regression algorithms entirely from scratch, and evaluate different modeling strategies on a real-world meteorological dataset.

---

## Learning Objectives

- Linear Regression
- Polynomial Regression
- Maximum Likelihood Estimation (MLE)
- Model Evaluation
- Feature Engineering
- Outlier Analysis
- Data Cleaning
- Noise Analysis
- Overfitting vs Underfitting

## Programming Topics

The implementation includes:

- Multivariate Linear Regression
- Polynomial Regression
- Closed-form Normal Equation
- Probabilistic Regression
- Gaussian Noise Analysis
- Feature Selection
- Correlation Analysis
- Outlier Detection
- Model Evaluation using MSE and MAE

---

## Implemented From Scratch

The assignment requires implementing all algorithms manually.

Machine learning libraries such as Scikit-Learn are intentionally not used.

Allowed libraries:

- numpy
- pandas
- matplotlib

---

## Learning Outcomes

After completing this assignment students should be able to

- derive the Normal Equation
- implement regression algorithms from scratch
- analyze overfitting
- evaluate regression models
- compare deterministic and probabilistic regression
- interpret learned parameters
