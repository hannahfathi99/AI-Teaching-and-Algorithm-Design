# Homework 01 – Uninformed Search Algorithms

## Overview

This assignment was designed for the undergraduate Artificial Intelligence course.

The objective is to strengthen students' understanding of classical uninformed search algorithms through both theoretical analysis and advanced programming implementations.

The programming section goes beyond standard textbook exercises and introduces dynamic environments, memory-efficient search techniques, adaptive depth strategies, and large-scale graph search problems.

---

## Topics Covered

- Depth First Search (DFS)
- Breadth First Search (BFS)
- Uniform Cost Search (UCS)
- Graph Search vs Tree Search
- State Pruning
- Memory-efficient Search
- Iterative Deepening DFS
- Dynamic Depth Adaptation
- Dynamic Uniform Cost Search
- Bidirectional Search
- Complexity Analysis

---

## Assignment Structure

### Part 1 — Theory

Students analyze

- DFS
- BFS
- UCS

including

- traversal order
- optimal path
- complexity analysis
- sparse vs dense graphs
- rational agents

---

### Part 2 — Programming

Students implement from scratch

- Advanced DFS with State Pruning

- Adaptive IDDFS

- Dynamic Uniform Cost Search

- Memory-efficient Bidirectional Search

using Python only.

No built-in graph search libraries are allowed.

---

## Learning Objectives

After completing this assignment students should be able to

- understand uninformed search algorithms

- compare graph search methods

- analyze complexity

- design scalable search algorithms

- implement memory-efficient search

- evaluate algorithm performance
